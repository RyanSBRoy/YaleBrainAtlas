# The Yale Brain Atlas

This project is an unofficial, work-in-progress, library for the [[[Yale Brain Atlas](https://yalebrainatlas.github.io/YaleBrainAtlas/atlas_viewer.html#model=images/Yale_Brain_Atlas.obj,images/Yale_Brain_Atlas.mtl)]]. It can be used for analysis and visualization of the brain, from the level of individual parcels to the whole cortex.
